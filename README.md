# E-Commerce Analytics Platform

A futuristic dark-mode e-commerce platform built with React, Vite, TypeScript, and Recharts.

## Features
- Product storefront with search and category filtering
- Cart experience with quantity controls and checkout celebration
- Analytics dashboard with revenue, top sellers, funnel, and category share charts
- Admin panel for product CRUD and order management
- Local state persisted to `localStorage`

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Notes
- The app uses `lucide-react`, `recharts`, and `canvas-confetti`.
- State persists across reloads using `localStorage`.
