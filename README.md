# 🚀 Cyber Bazaar

A modern full-stack futuristic e-commerce platform built using the MERN stack with a cyberpunk-inspired UI. Cyber Bazaar provides a complete online shopping experience with user authentication, product management, real-time analytics, secure checkout simulation, order management, and an admin dashboard.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/je-suis-Abhinav/Cyber-Bazaar.git
```

Move into the project

```bash
cd Cyber-Bazaar
```

Install frontend dependencies

```bash
npm install
```

Install backend dependencies

```bash
cd server
npm install
```

---

## ▶️ Running the Project

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
npm run dev
```

The application will be available at

```
http://localhost:4173
```

Backend runs on

```
http://localhost:5000
```
